#URL SHORTENER
A URL shortener with user authentication, click analytics, and Redis caching. Built with Spring Boot 3 (Java 17), PostgreSQL, Redis on the backend and Next.js 15 / React 19 / Tailwind on the frontend.

To run: start the backend with ./mvnw spring-boot:run from backend/url-shortener-backend (requires Postgres + Redis), and the frontend with npm run dev from frontend/url-shortener-frontend.
